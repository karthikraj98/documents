Update your system: PG-ADMIN ------UBUNTU

Install dependencies: Ensure that the required dependencies are installed:

sudo apt install -y curl ca-certificates

Add the pgAdmin repository: Add the PostgreSQL Global Development Group (PGDG) repository, which includes the pgAdmin 4 package.

curl https://www.pgadmin.org/static/packages_pgadmin_org.pub | sudo tee /etc/apt/trusted.gpg.d/packages_pgadmin_org.asc
echo "deb https://ftp.postgresql.org/pub/pgadmin/pgadmin4/apt/$(lsb_release -c | awk '{print $2}') pgadmin4 main" | sudo tee /etc/apt/sources.list.d/pgadmin4.list

Update the package list:

sudo apt update

Install pgAdmin 4: Now, you can install pgAdmin 4 with the following command:
sudo apt install pgadmin4
